The HTML and CSS Basic questions

  1. How do you set up your development environment for HTML and CSS?
  2. There are three ways by which one can go through the following and can proceed upto for the same:

    • Text Editor or IDE:

      • A basic program for writing plain text, often with features like syntax highlighting and code
      completion for HTML and CSS. Popular options include Visual Studio Code (VS Code), Sublime
      Text, and Atom (all free and open-source).

    • Web Browser:

      • You'll need a web browser to preview your website as you code. Common choices include Chrome
      , Firefox, Safari, or Edge.

    • Optional: Local Web Server (for convenience):

      • A local web server allows you to run your HTML and CSS files on your computer like a real website.
      This can be helpful for testing and development. Many code editors, like VS Code, have built-in live
      server extensions you can install.

  3. How do you create a basic HTML document structure?

    4. DOCTYPE Declaration:
    This line specifies the document type as HTML. It's the very first line of your code and tells the browser how to interpret the code


    HTML Element:
    This is the root element of your document and encompasses everything else.

    Head Element: This section contains metadata about your document, but it's not displayed on the webpage itself. It includes the document title, character encoding, and links to external resources like CSS stylesheets.



    Body Element: This section contains the content that will be displayed on the webpage. This is where you'll put your headings, paragraphs, images, and other elements that make up the visible part of your website.

  4. How do you link a CSS file to an HTML file ?

    5. You can link a CSS file to an HTML file using the link tag placed within the head section of your HTML document. The link tag has a few important attributes for specifying the connection:

    • rel: This attribute defines the relationship between the HTML document and the linked file. In the case of CSS, you set it to "stylesheet".
    • type: This attribute specifies the type of the linked file. For CSS, you set it to "text/css".
    • href: This attribute specifies the location (path) of the CSS file relative to your HTML file.

  5. How can you style text and elements using CSS?

    6. Cascading Style Sheets (CSS) is a powerful tool for web developers to define the presentation of a web page's content, including text and various elements. Here's how you can
    achieve stunning visual effects and control the layout using CSS:

    1. Text Styling:

      2. Font Properties:
      font-family: This property dictates the typeface used for the text. You can specify a specific font name like 'Arial' or generic categories like 'serif' or 'sans-serif'.
      font-size: This controls the size of the text, and you can define it in various units like pixels (px), points (pt), or even relative units like ems.
      font-weight: This alters the boldness of the text, with values like 'normal' for regular weight and 'bold' for bolder text.
      font-style: Use this property to make the text italic or oblique with values like 'italic' or 'oblique'.

      Text Decoration:
      Text-decoration: This property allows you to add decorative lines to text, including underline, overline, line-through, or none.
      Text Alignment: Text-align: This property controls how text is positioned within its container element. You can use values like 'left', 'center', 'right', or 'justify' for full justification.

    2. Element Styling:

      3.  Background
      background-color: This property defines the background color of an element. You can use color names, hexadecimal codes (e.g., #ff0000 for red), or RGB values.
      background-image: Set a background image for the element using a URL path to the image.

      Borders:
      border: This shorthand property defines all border styles at once, including width, color, and style (solid, dashed, dotted).

      Dimensions:
      width: This property sets the width of the element. height: This property sets the height of the element.

      Padding & Margin:
      padding: This property defines the space between the element's border and its content.
      margin: This property defines the space between the element and surrounding elements.
